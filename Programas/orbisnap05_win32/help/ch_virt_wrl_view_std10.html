Orbisnap Interface

This topic describes the following components of the Orbisnap interface:


  • Menu Bar
  • Toolbar
  • Navigation Panel

Using the menus, toolbar, and navigation panel, you can perform many of the same operations on the virtual world that you can with the Virtual Reality Toolbox viewer. For further details on using this interface open your Web browser and go to:

http://www.mathworks.com/access/helpdesk/help/toolbox/vr/ch_virtual_world_viewing.html

You can also download the PDF version of the Virtual Reality Toolbox User's Guide documentation from the MathWorks Web site:

http://www.mathworks.com/access/helpdesk/help/pdf_doc/vr/vr.pdf
